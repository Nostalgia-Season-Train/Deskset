## 简介
数字桌搭是一款专注整合笔记应用的桌面美化软件，旨在优化笔记的读写管理流程，从而提高个人生产力。


## 亮点
- 深度集成笔记应用
- Vue3 + FastAPI 前后端分离架构，方便二次开发
- Tauri + PyStand 打包构建，相较同类产品启动迅速且占用小巧
- 组件可以自动隐藏，保持桌面干净整洁
- 轻松保存和应用主题，并通过 CSS 定制样式
