# 开发概览

数字桌搭采用 Tauri + Vue 的前后端架构，支持开发者编写自己的部件和插件。

## 技术栈

| 层 | 技术 | 说明 |
|----|------|------|
| 桌面壳 | Tauri (Rust) | 窗口管理、系统集成 |
| 前端 | Vue 3 + Vite | 桌面窗口、管理窗口、部件 |
| 后端 | FastAPI | 数据中心、自动化引擎 |

## 项目结构

```
Deskset/
  100-Site/        官网（VitePress）
  200-Vite/        前端源码（Vue + Vite）
  300-Tauri/       桌面壳源码（Rust + Tauri）
  400-Build/       构建脚本
```

## 开发什么

- **部件（Widget）**：用 Vue 编写的桌面小组件，如时钟、硬件监控等
- **插件（Plugin）**：包含前端部件和后端路由的扩展包
